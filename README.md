# Multer Todo App

A full-stack Todo application with file upload feature using Multer.

## Features
- Add tasks
- Upload image with task
- Delete & update tasks
- Image storage using Multer

## Tech Stack
- Node.js
- Express.js
- Multer
- HTML, CSS, JavaScript

## How to Run
1. npm install
2. node server.js
3. Open browser at http://localhost:5000

## Note
This project demonstrates file upload handling in Node.js using Multer.
